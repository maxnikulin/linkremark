LinkRemark Test: JSON-LD metadata in multiple scripts

This page contains several ld+json <script> elements: Organization, WebPage, and Article.

Author, datePublished, url, publisher, image may be extracted if article is considered as the primary entry. Title have to be taken from the head element.

Peculiarities:

  • @id attributes are anchors on the top of the site (empty paths).
  • Article name is an empty string.
  • WebPage name is site name, not article and page title.
  • Article and WebPage uses the same url, so WebPage name is really confusing.
  • There is no cross-references between records, so primary one have to be guessed.
