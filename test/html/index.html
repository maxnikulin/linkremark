LinkRemark Test Pages

Collection of HTML pages with special markup to check if LinkRemark could reasonably handle such cases.

Metadata

  • Link elements with rel="alternate" and media, hreflang, type, title attributes
  • ld+json <script> element
  • Some basic metadata in html head element
  • Special cases of metadata in html head element: relative URLs, insane number of elements.
  • Microdata embeded in HTML (itemscope, itemprop)
  • Microdata - author
  • Microdata: Product
  • Microdata for QAPage
  • Page without title element and other metadata
  • Page without metadata, link with some query parameters and anchor to make URL long
  • HTML entities in metadata
  • DOI in <meta>

Links

  • Various URLs (unicode, data:, javascript:, etc.
  • Canonical link with missed trailing slash

Images

  • Image variants

Selection

  • Styles to hide elements or control white space, forms, etc.
  • Long text to check length limit

PDF File with nested <iframe> elements

  • Page with PDF file embedded using a single nested iframe
